📖 Visão Geral do Sistema

O Sistema Piaget - Gestão Acadêmica é uma aplicação web completa desenvolvida com tecnologias modernas (HTML5, CSS3, JavaScript ES6+) e arquitetura orientada a objetos. O sistema oferece uma solução integrada para gerenciamento de escolas, professores, alunos, finanças e relatórios, com foco em usabilidade e performance.

🎯 Objetivos do Sistema

  • Centralizar a gestão acadêmica em uma única plataforma
  • Fornecer ferramentas de análise e tomada de decisão
  • Automatizar processos administrativos
  • Garantir segurança e persistência dos dados
  • Oferecer interface intuitiva e responsiva

✨ Principais Funcionalidades

🏫 Gestão de Escolas

Cadastro, edição, exclusão e busca de instituições de ensino

👨‍🏫 Gestão de Professores

Controle completo do corpo docente com vínculo às escolas

👨‍🎓 Gestão de Alunos

Matrículas, notas, frequência e desempenho acadêmico

💰 Módulo Financeiro

Controle de receitas, despesas e fluxo de caixa

📊 Dashboard Analítico

KPIs, gráficos interativos e dados em tempo real

📑 Relatórios Customizáveis

Exportação para PDF e Excel com templates

🔐 Autenticação de Usuários

Sistema de login seguro com persistência de sessão

📧 Notificações por Email

Sistema de alertas e comunicação integrada

🏗️ Classes e POO

Class Auth

class Auth {
    - users: Array          // Lista de usuários cadastrados
    - currentUser: Object   // Usuário logado atualmente
    
    + login(email, password): boolean      // Realiza login
    + logout(): void                       // Encerra sessão
    + getCurrentUser(): Object             // Retorna usuário atual
    + isAuthenticated(): boolean           // Verifica autenticação
}

Class Database

class Database {
    - escolas: Array        // Lista de escolas
    - professores: Array    // Lista de professores
    - alunos: Array         // Lista de alunos
    
    + carregarDados(): void                    // Carrega dados do LocalStorage
    + salvar(): void                           // Persiste dados
    + getStats(): Object                       // Retorna estatísticas
    + carregarExemplos(): void                 // Carrega dados de exemplo
}

Class Financeiro

class Financeiro {
    - transacoes: Array     // Lista de transações
    
    + carregarTransacoes(): Array              // Carrega transações
    + salvar(): void                           // Persiste transações
    + adicionar(t): Object                     // Adiciona nova transação
    + getSaldo(): number                       // Calcula saldo
    + getReceitasTotal(): number               // Soma receitas
    + getDespesasTotal(): number               // Soma despesas
}

Class NotificationSystem

class NotificationSystem {
    - notificacoes: Array   // Lista de notificações
    
    + adicionar(titulo, msg, tipo): Object     // Cria notificação
    + mostrarToast(notif): void                // Exibe toast
    + getNaoLidas(): Array                     // Retorna não lidas
    + marcarComoLida(id): void                 // Marca como lida
}

🚀 Guia de Uso Passo a Passo

Passo 1: Acesso ao Sistema

1. Abra o arquivo index.html em qualquer navegador moderno

2. Utilize as credenciais de acesso:

  • Email: admin@academico.com
  • Senha: admin123

Passo 2: Cadastro de Escolas

1. Clique no menu "Escolas" na sidebar

2. Preencha o formulário com os dados da instituição

3. Clique em "Cadastrar Escola"

4. A escola aparecerá na lista abaixo

Passo 3: Cadastro de Professores

1. Clique no menu "Professores"

2. Selecione a escola no campo correspondente

3. Preencha os dados do professor

4. Clique em "Cadastrar Professor"

Passo 4: Cadastro de Alunos

1. Clique no menu "Alunos"

2. Selecione a escola e informe a turma

3. Preencha os dados do aluno

4. Opcionalmente, adicione uma nota inicial

5. Clique em "Cadastrar Aluno"

Passo 5: Gerenciamento Financeiro

1. Acesse o menu "Financeiro"

2. Registre receitas e despesas

3. Visualize o extrato completo

4. Exporte os dados para Excel

Passo 6: Visualização do Dashboard

1. Acesse o menu "Dashboard" (Piaget Relatórios)

2. Visualize os KPIs atualizados

3. Analise os gráficos interativos

4. Acompanhe os dados em tempo real via WebSocket simulado

Passo 7: Geração de Relatórios

1. Acesse o menu "Relatórios"

2. Selecione o tipo de relatório desejado

3. Clique em "Gerar PDF" ou "Exportar Excel"

4. O arquivo será baixado automaticamente

Passo 8: Notificações

1. Acesse o menu "Notificações"

2. Visualize as notificações recebidas

3. Clique em "Nova Notificação" para criar alertas

4. Marque como lida as notificações já visualizadas

☁️ Guia de Deploy

Opção 1: Hospedagem Estática (Recomendado)

1. Organize os arquivos na estrutura:
   - index.html
   - style.css
   - script.js

2. Acesse uma das plataformas gratuitas:
   - Vercel: vercel.com
   - Netlify: netlify.com
   - GitHub Pages: pages.github.com

3. Faça upload dos arquivos ou conecte o repositório

4. O sistema estará disponível online em minutos

Opção 2: Servidor Local

# Usando Python (Windows/Mac/Linux)
python -m http.server 8000

# Usando Node.js (npm install -g http-server)
http-server -p 8000

# Usando PHP
php -S localhost:8000

# Acesse: http://localhost:8000

Requisitos de Hospedagem

  • ✅ Navegador moderno (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
  • ✅ Suporte a LocalStorage (todos os navegadores modernos)
  • ✅ JavaScript ES6+ habilitado
  • ✅ Conexão com internet (para CDNs e bibliotecas externas)

Bibliotecas Externas Utilizadas (CDN)
